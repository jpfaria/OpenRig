Regras do sistema visual
1

Shell consistente

Todos os blocos compartilham grade, proporção, título e zona de ação.

2

Tipo diferencia o visual

Drive, amp, mod, delay, cab e utility mudam material, altura e densidade de knobs.

3

Desktop com knobs de verdade

No desktop os ajustes principais aparecem como knobs, não só sliders e inputs.

OK

Essa linguagem de blocos está certa

Shell compartilhável, mas cada tipo de bloco parece um pedal/módulo de gear de verdade.

A

Precisa ajustar o nível de “pedal”

O caminho está certo, mas quer mais literalidade de hardware ou mais sobriedade técnica.
