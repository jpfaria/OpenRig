# OpenRig Brand Direction

This file exists to keep the visual direction consistent.

## References

Use these only as directional cues, not as templates to copy:

- Fender
- Fender Studio
- Spotify

## Desired Qualities

- compact logo, not a banner
- strong icon plus clean wordmark
- immediate readability
- modern but musical
- simple enough to work on README, app, plugin, and hardware
- should suggest pedalboard / signal / control, without looking technical

## Avoid

- diagrams
- subtitles inside the logo
- giant horizontal hero treatment pretending to be a logo
- too many tiny details
- startup-generic gradients without meaning
- marks that look like dashboards or organograms

## Product Feeling

- premium music tool
- friendly to musicians
- stage-ready
- studio-ready
- modern, but not sci-fi
