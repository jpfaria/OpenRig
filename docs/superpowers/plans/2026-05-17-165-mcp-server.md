# MCP Server Implementation Plan (#165)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add an opt-in MCP server that lets any MCP client drive the live OpenRig instance through the existing command bus, with full functional parity with the GUI.

**Architecture:** A `Send` bridge in the `application` crate connects the async `rmcp` server (its own thread) to the `!Send` `LocalDispatcher` on the frontend thread: requests flow over `std::sync::mpsc`, replies over `futures::channel::oneshot`, drained on a Slint timer tick. `adapter-mcp` is a frontend-agnostic library; tool schemas are auto-derived from the `schemars`-derived `Command` schema (one tool per variant). The GUI gains a `--mcp[=addr]` flag that spawns the server and drains the bridge.

**Tech Stack:** Rust, `rmcp` 1.7.0 (Streamable HTTP), `schemars` 0.8, `serde_json`, `futures` (oneshot), `tokio`, Slint `Timer`.

**Spec:** `docs/superpowers/specs/2026-05-17-165-mcp-server-design.md`

**Workspace:** all work in `.solvers/issue-165` on branch `feature/issue-165` (from `feature/issue-436`). Stage explicit paths only — never `git add -A`. `./scripts/qa.sh` green before every push; comment on issue #165 after every push.

```rust
//! `Send` bridge between an async transport (MCP/gRPC) and the `!Send`
//! `LocalDispatcher`. The transport thread `submit`s a `Command`; the
//! frontend thread `drain`s and dispatches on its own thread, replying
//! over a `futures` oneshot. No tokio runtime is pulled into this crate.

use std::sync::mpsc::{self, Receiver, Sender, TryRecvError};

use anyhow::Result;
use futures::channel::oneshot;

use crate::command::Command;
use crate::dispatcher::CommandDispatcher;
use crate::event::Event;

/// Result of one dispatched command: `Ok(events)` or a stringified error
/// (the bridge crosses a thread boundary; `anyhow::Error` is not `Send`-safe
/// to forward as-is to the transport's serialization layer).
pub type DispatchOutcome = Result<Vec<Event>, String>;

struct BridgeRequest {
    cmd: Command,
    reply: oneshot::Sender<DispatchOutcome>,
}

/// Cloneable, `Send` handle held by the transport (MCP server thread).
#[derive(Clone)]
pub struct CommandBridge {
    tx: Sender<BridgeRequest>,
}

impl CommandBridge {
    /// Queue a command. Returns a oneshot receiver that resolves once the
    /// frontend drains and dispatches it. Never blocks.
    pub fn submit(&self, cmd: Command) -> oneshot::Receiver<DispatchOutcome> {
        let (reply, rx) = oneshot::channel();
        // If the frontend is gone the receiver will simply never resolve;
        // the transport layer applies its own request timeout.
        let _ = self.tx.send(BridgeRequest { cmd, reply });
        rx
    }
}

/// Receiver side, owned by the frontend thread.
pub struct BridgeDrain {
    rx: Receiver<BridgeRequest>,
}

impl BridgeDrain {
    /// Dispatch up to `cap` queued commands on the calling (frontend) thread.
    /// Returns how many were handled. Non-blocking; safe to call every tick.
    pub fn drain(&self, dispatcher: &dyn CommandDispatcher, cap: usize) -> usize {
        let mut handled = 0;
        while handled < cap {
            match self.rx.try_recv() {
                Ok(req) => {
                    let outcome = dispatcher
                        .dispatch(req.cmd)
                        .map_err(|e| e.to_string());
                    let _ = req.reply.send(outcome);
                    handled += 1;
                }
                Err(TryRecvError::Empty) | Err(TryRecvError::Disconnected) => break,
            }
        }
        handled
    }
}

/// Create a connected `(transport handle, frontend drain)` pair.
pub fn channel() -> (CommandBridge, BridgeDrain) {
    let (tx, rx) = mpsc::channel();
    (CommandBridge { tx }, BridgeDrain { rx })
}
```

```rust
//! Bridges the `schemars`-derived schema of [`crate::command::Command`] into
//! a per-variant tool surface. Single source of truth = the `Command` enum;
//! no hand-written JSON Schema.

use schemars::schema_for;
use serde_json::Value;

use crate::command::Command;

/// All `Command` variant names, derived from the externally-tagged schema.
pub fn command_variant_names() -> Vec<&'static str> {
    // `schemars` 0.8 emits an externally-tagged enum as `oneOf`, each entry an
    // object with a single `required` key = the variant name (struct variants)
    // or an enum-of-one `const`/`enum` for unit variants. We derive the names
    // once from the static schema and intern them.
    static NAMES: std::sync::OnceLock<Vec<String>> = std::sync::OnceLock::new();
    NAMES
        .get_or_init(|| extract_variant_names(&serde_json::to_value(schema_for!(Command)).unwrap()))
        .iter()
        .map(|s| -> &'static str { Box::leak(s.clone().into_boxed_str()) })
        .collect()
}

fn extract_variant_names(root: &Value) -> Vec<String> {
    let mut out = Vec::new();
    if let Some(one_of) = root["oneOf"].as_array() {
        for entry in one_of {
            // Struct variant: { "type":"object", "required":["VariantName"], ... }
            if let Some(req) = entry["required"].as_array() {
                if let Some(name) = req.first().and_then(Value::as_str) {
                    out.push(name.to_string());
                    continue;
                }
            }
            // Unit variant: { "type":"string", "enum":["VariantName"] }
            if let Some(en) = entry["enum"].as_array() {
                if let Some(name) = en.first().and_then(Value::as_str) {
                    out.push(name.to_string());
                }
            }
        }
    }
    out
}

/// Object schema for a single variant's arguments (the value side of the
/// externally-tagged pair). Unit variants get an empty object schema.
pub fn command_variant_schema(variant: &str) -> Value {
    let root = serde_json::to_value(schema_for!(Command)).unwrap();
    if let Some(one_of) = root["oneOf"].as_array() {
        for entry in one_of {
            let is_match = entry["required"]
                .as_array()
                .and_then(|r| r.first())
                .and_then(Value::as_str)
                == Some(variant)
                || entry["enum"]
                    .as_array()
                    .and_then(|e| e.first())
                    .and_then(Value::as_str)
                    == Some(variant);
            if is_match {
                // Struct variant: properties.<Variant> is the args object.
                if let Some(args) = entry["properties"].get(variant) {
                    return args.clone();
                }
                break;
            }
        }
    }
    serde_json::json!({ "type": "object", "properties": {}, "required": [] })
}

/// `SetBlockParameterNumber` -> `set_block_parameter_number`.
pub fn tool_name(variant: &str) -> String {
    let mut s = String::with_capacity(variant.len() + 8);
    for (i, ch) in variant.char_indices() {
        if ch.is_uppercase() && i != 0 {
            s.push('_');
        }
        s.push(ch.to_ascii_lowercase());
    }
    s
}

/// Reverse of [`tool_name`]; `None` if it matches no `Command` variant.
pub fn variant_from_tool_name(tool: &str) -> Option<&'static str> {
    command_variant_names()
        .into_iter()
        .find(|v| tool_name(v) == tool)
}
```

```rust
//! One MCP tool per `Command` variant. Schema from `application::command_schema`
//! (single source of truth). `call_tool` rebuilds the externally-tagged
//! `Command` JSON (`{ "<Variant>": <args> }`), deserializes, and submits to
//! the bridge.

use anyhow::Result;
use application::bridge::CommandBridge;
use application::command::Command;
use application::command_schema::{
    command_variant_names, command_variant_schema, tool_name, variant_from_tool_name,
};
use application::event::Event;
use serde_json::{json, Value};

/// `(tool_name, description, input_schema)` for every variant.
pub fn tool_specs() -> Vec<(String, String, Value)> {
    command_variant_names()
        .into_iter()
        .map(|v| {
            (
                tool_name(v),
                format!("OpenRig command: {v}"),
                command_variant_schema(v),
            )
        })
        .collect()
}

/// Map an incoming tool call to a `Command`, submit, await events.
pub async fn dispatch_tool(
    bridge: &CommandBridge,
    tool: &str,
    args: Value,
) -> Result<Vec<Event>> {
    let variant = variant_from_tool_name(tool)
        .ok_or_else(|| anyhow::anyhow!("unknown tool: {tool}"))?;
    // Reconstruct externally-tagged form serde expects for `Command`.
    let tagged = json!({ variant: args });
    let cmd: Command = serde_json::from_value(tagged)
        .map_err(|e| anyhow::anyhow!("invalid arguments for {tool}: {e}"))?;
    let rx = bridge.submit(cmd);
    rx.await
        .map_err(|_| anyhow::anyhow!("frontend dropped the bridge"))?
        .map_err(|e| anyhow::anyhow!(e))
}
```
