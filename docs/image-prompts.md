# OpenRig Image Prompts

This file contains production prompts for generating OpenRig marketing images in tools such as Midjourney, Flux, DALL·E, or similar image models.

The target visual language is:

- premium guitar gear
- pro audio product photography
- modern hardware startup branding
- realistic instruments, screens, pedals, and hands
- cinematic but believable lighting

## Global Art Direction

Use these constraints across all prompts:

- premium pedalboard hardware
- realistic guitar and instrument details
- realistic smartphone and tablet proportions
- elegant industrial design
- black graphite chassis with subtle brushed metal
- warm amber and cool cyan accent lighting
- high-end studio and stage aesthetic
- clean typography on screens
- no generic sci-fi UI
- no cartoon look
- no flat infographic look

Recommended negatives:

- cartoon
- low poly
- wireframe
- infographic
- diagram
- isometric illustration
- distorted hands
- extra fingers
- broken guitar anatomy
- warped phone
- fake unreadable UI clutter
- toy-like pedal
- cheap plastic look

## Practical Notes

- Generate wide versions first for README and site headers.
- Generate portrait versions for social and app store style mockups.
- Keep the hardware shape consistent across generations.
- Prefer one consistent industrial design language before generating many scenes.
- If the first generations look too futuristic, reduce the words `futuristic` and `sci-fi` entirely.
- If the UI looks fake, reduce screen detail and ask for `minimal believable music control interface`.
