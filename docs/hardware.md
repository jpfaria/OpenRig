# OpenRig Hardware

OpenRig hardware is the live version of the platform.

## The Idea

Instead of every musician carrying a separate isolated unit, OpenRig hardware is meant to work as one central stage device that can accept multiple instruments and still let each player control their own sound.

## In Practice

- guitar, bass, vocals, keys, or other inputs can connect to the same unit
- each musician gets one or more dedicated chains
- each track has its own sound chain and state
- each musician controls only their own track from a phone or tablet
- the audio stays in the hardware, not in the mobile device

## Why This Matters

- one hardware brain on stage
- less duplicated gear
- personal control without exposing the whole rig to everyone
- one product experience across rehearsal, stage, and studio

## Product Direction

This hardware mode is part of the same OpenRig platform as:

- standalone on computer
- VST3 inside a DAW
- server mode for remote control
- dedicated hardware for live use

The goal is one OpenRig experience across all of them.
